## Project Overview

### **Problem**
- 기록의 장벽: 기존 다이어리 서비스의 높은 진입 장벽과 백지 공포증
- 단편적 기록: 텍스트 위주의 단조로운 기록과 정서적 보상의 부재

### **Solution**
- **Chronological Structure**: 아침/오후/저녁 시간대별 활동 태그와 답변 유도형 질문을 통한 기록 장벽 최소화
- **Hybrid AI Pipeline**
- **KoBERT**: 47종 미세 감정 지표를 통한 정교한 심리 분석 및 정량화
- **LLaMA**: 파편화된 태그 데이터를 서사적인 일기 문장으로 자동 변환
- **SD3 (LoRA)**: 분석된 감정을 우리만의 독자적 화풍으로 시각화하여 정서적 카타르시스 제공
- **Today's Recommendation**: 감정 수치(-1.0 ~ +1.0) 기반의 가벼운 제안을 통한 마이크로 케어
