# 감정서가
LLM & Diffusion 모델 기반의 다차원 감정 분석 그림일기 서비스

### Tech Stack
- Frontend: Kotlin (Android, Jetpack Compose)
- Backend: Spring Boot 3.4, FastAPI
- AI: GPT-4o-mini, Stable Diffusion 3
- DB: PostgreSQL 16, Pinecone
