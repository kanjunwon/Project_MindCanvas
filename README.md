## Project Overview

### **Problem**
- 기존 다이어리 서비스의 높은 기록 장벽
- 단순 텍스트 위주의 단편적인 감정 기록의 한계

### **Solution**
- **LLM(GPT-4o-mini)**: 사용자의 키워드를 풍부한 서사로 확장하여 기록의 부담 완화
- **Diffusion(SD3)**: 분석된 감정을 시각화하여 정서적 보상 및 기록의 재미 극대화
- **Multi-Dimensional Analysis**: 47종의 미세 감정 지표를 통한 정교한 심리 상태 추적
